Sidebar layout — how it works

Left sidebar = your pipeline library. Main panel = Design or Run depending on what you select.

Which screen layout feels right for the main panel?

A — Design-first

New pipeline starts in Design. Run is a button inside the design view once it's saved.

PIPELINES
+ New Pipeline
🎨 Code Reviewer
🎨 Data Pipeline
🎨 Space Planner
🎨 Design: Code Reviewer
▶ Run this
Agent 1: Fetch
Agent 2: Analyze
Agent 3: Report
DAG visualization

B — Mode toggle per pipeline

Each pipeline in the sidebar has two icons: pencil (Design) and play (Run). Click the icon to open in that mode.

PIPELINES
+ New Pipeline
Code Reviewer ✏️ ▶
Data Pipeline ✏️ ▶
Space Planner ✏️ ▶
⚡ Run: Space Planner
SPECIFIC TASK
Plan a Mars sample return mission
● Agent 1: Mission Scope — running...
○ Agent 2: Resource Calc — waiting

My recommendation: B. Having ✏️ and ▶ per pipeline is clearer — you always know you're either editing the design or executing a run. No ambiguity about which mode you're in.
