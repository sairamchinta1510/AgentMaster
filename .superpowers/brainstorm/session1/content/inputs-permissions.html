One question

For permissions like GitHub — how should the connection work at run time?

A

OAuth / "Connect" button

Click "Connect GitHub" → opens auth flow in browser. Token stored in app for this session.

B

Paste a token / API key

User pastes a Personal Access Token or API key into a masked input field. Simpler, no redirect.

C

Both — designer chooses per permission

At design time, designer marks each permission as "OAuth" or "API key". Run time shows the right UI automatically.
