What should AgentMaster feel like?

The core question before we redesign anything

Current problem

Right now everything happens in one messy flow — you type an objective, LLM agents start designing and running at the same time, and there's no way to save or reuse anything. It feels like a black box.

What you described — two distinct modes

🎨 DESIGN TIME

Goal: Build a reusable agent pipeline for a class of problems


→ Describe what kind of work you want (e.g. "code review", "space mission planning")

→ AI designs the agents, their roles, how they connect

→ You review, tweak, approve each agent

→ The pipeline is saved to a Library


Think: building a tool, not using it

⚡ RUN TIME

Goal: Pick a saved pipeline and run it on a specific task


→ Browse your Library of saved pipelines

→ Load one (e.g. "space mission planner")

→ Provide the specific input ("Plan a Mars sample return mission")

→ Watch agents execute, stream results live


Think: using the tool you built

First question

Which of these best describes how you picture switching between the two modes?

A

Tab switcher at the top

One app, two tabs — "Design" and "Run". Always visible, easy to switch.

B

Separate screens / landing page

Home screen asks: "Design a new pipeline" or "Run an existing one". Each leads to its own full-screen experience.

C

Sidebar + main panel

Left sidebar lists your saved pipelines. Click one to either edit (Design) or run (Run). Mode determined by context.
